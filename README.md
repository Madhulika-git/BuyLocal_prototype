Project Overview
----------------
BuyLocal is a web-based platform designed to help local shop owners digitally showcase their businesses and allow customers to easily discover nearby shops, products, and offers.
The platform bridges the gap between small businesses and customers by providing an accessible and affordable digital presence.

Key Features
-------------
Shop registration and management
Category-based shop browsing
Search by shop name or product
Real-time nearby shop detection (GPS-based)
Image uploads for products and sales
Google Maps navigation integration
User authentication
Firebase Firestore database integration

Tech Stack
-----------
HTML
CSS
JavaScript (ES6 Modules)
Firebase Firestore
Firebase Hosting (GitHub Pages for deployment)

Impact
-------
Helps small businesses expand digital reach
Improves product discoverability for customers
Promotes local economic growth
